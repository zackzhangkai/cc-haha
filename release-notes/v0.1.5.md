# Claude Code Haha v0.1.5

这是一个以桌面端稳定性、兼容性和响应速度优化为主的版本。

相比 `v0.1.4`，这次主要修复了 WebFetch 在第三方提供商下失败、`@` 文件列表异常、跨会话 TODO 串台、权限模式切换后会话异常，以及桌面端首字返回过慢等问题。

## 主要更新

- 优化了 WebFetch 在第三方提供商下的可用性，桌面端默认跳过容易失败的预检查，并补齐了 HTML 转 Markdown 依赖。
- 优化了桌面端首字返回速度，减少第三方模型和代理链路下“发出去很久才开始回复”的问题。
- 修复了跨会话 TODO 串台，以及已完成任务在下一轮对话中残留的问题。
- 修复了 `@` 文件引用在部分目录下拿不到文件列表的问题，并补上了更明确的错误提示。
- 修复了权限模式切换后，旧 CLI 进程误伤新会话，导致会话异常退出的问题。
- 优化了标签拖拽、项目筛选、AskUserQuestion 流程和工具消息渲染体验。
